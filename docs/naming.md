# Product name

The public product name is **Cowlick**.

A cowlick is a lock of hair that falls over the forehead. The name fits a small utility that grows naturally from the MacBook's camera housing without describing the UI literally or borrowing Apple's Dynamic Island branding.

The name was checked on 2026-07-18 against GitHub, Homebrew formulae and casks, and general web/software results. An established, unrelated open-source package manager already uses the exact name at [cowlick/cowlick](https://github.com/cowlick/cowlick). The maintainer selected Cowlick with that conflict understood; this project does not claim the name is globally unique and is not affiliated with that repository.

Cowlick is an unofficial community project. It is not affiliated with, endorsed by, or sponsored by OpenAI or Apple.
