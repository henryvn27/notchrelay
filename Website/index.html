Allow once means this request. Nothing else.

Every approval is matched to one pending UUID. Cowlick never makes Allow the default and never turns a timeout, malformed response, or disconnected app into permission.

  • Authenticated locally A current request arrives over private IPC.
  • Explicit every time Allow once or Deny is a deliberate action.
  • Failure defers Codex keeps its normal approval UI.
Read the threat model ↗

Prompts stay private.

Prompt previews are off by default. Cowlick has no analytics, Cowlick account, cloud backend, ads, or third-party crash reporter.

Lifecycle fields only. Local observation ignores prompt, message, command, tool-input, and result payloads. Approvals come only from authenticated hooks.

No prompt history. Full prompts and approval operations are not persisted.

No hidden aggregation. Provider accounts and cost coverage remain separate.

cd cowlick
brew install xcodegen
./Scripts/install_local.sh
