## 1.0.0 - 2026-07-18

- Native notch-aware and non-notch island for Codex working, approval, completed, failed, and multi-session states.
- Exact request-matched Allow once and Deny actions with safe Codex fallback.
- Authenticated private Unix-domain socket and bundled Swift helper.
- Idempotent, reversible Codex hook onboarding that preserves unrelated configuration.
- Trust-first integration diagnostics and executable validation for installed Codex app and CLI variants.
- Restart recovery that keeps ledger entries inspectable as unconfirmed without inflating active-session state.
- Used or remaining quota display with an even-pace marker and a time-to-empty forecast when enough data exists.
- Multiple separately labeled OpenAI API and Anthropic API organization-billing accounts with Keychain credentials and month-to-date actual costs.
- Optional native Caps Lock signaling with original-state restoration.
- Menu bar, Settings, onboarding, diagnostics, launch-at-login, Sparkle integration, universal packaging workflows, CI, and public documentation.
