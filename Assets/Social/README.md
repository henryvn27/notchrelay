# Cowlick social assets

- `github-social-preview.png` is the 1280 × 640 repository social preview.
- `x-launch.png` is the 1600 × 900 launch image.
- `launch-copy.md` contains factual, channel-specific release copy.

Both images use current Cowlick artwork and realistic non-sensitive demo data captured from the app on a 2× non-notch display. The X image is an intentional multi-state launch composition rather than a duplicate of the README hero. Neither image claims physical notch verification or includes an installation command. Add the verified command to post copy only after it succeeds against the public tap, then compare the shown UI with the final signed build.
