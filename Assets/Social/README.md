# Cowlick social assets

- `github-social-preview.png` is the 1280 × 640 repository social preview.
- `x-launch.png` is the 1600 × 900 launch image.
- `launch-copy.md` contains factual, channel-specific release copy.

Both images use current Cowlick artwork and realistic non-sensitive demo data. They also include the intended v1.0 Homebrew command, so they are release-candidate assets rather than evidence that installation works today. Before publishing or uploading them to GitHub, verify the command against the public tap and compare the shown island with the final released build.

Cowlick is an unofficial community project and is not affiliated with or endorsed by OpenAI.
